#Langton's Ant

An implementation of the famous Langton's ant in SDL.

##Compile

`make`

##Running

Run with `./langtonsant`.

Controls:

* F11: Toggle fullscreen
* Q: Quit
* ESC: If in fullscreen, come out of it; else quit
* C: Centre on ant
* +/- and scroll wheel: Zoom
* Click and drag: pan around
* P: Pause
